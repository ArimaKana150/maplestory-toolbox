# 新楓之谷 工具箱

專為新楓之谷（台服）玩家打造的線上工具箱，純前端、免安裝、開網頁即用。

**線上使用：** https://arimakana150.github.io/maplestory-toolbox/

## 功能

- **VIP 階級計算機** — 輸入目前累積的 VIP 點數、選定想達到的階級，立即算出還差多少點數與所需消費金額（購買／送禮）。
- **角色穿搭預覽** — 線上預覽髮型、臉型、帽子、上衣、連身、下身、鞋子、武器，支援膚色與單品染色（色相），每件道具直接顯示道具代碼。

## 資料來源

- VIP 階級規則依官方公告製作，實際以官方公告為準。
- 穿搭道具資料來自 [maplestory.io](https://maplestory.io) 公開 API（台服 TWMS）。

## 技術

純前端靜態網頁（HTML / CSS / JavaScript），無需後端，部署於 GitHub Pages。
